# MultiplayerGameOfLife
A clone of Conway's Game of Life in NodeJS and HTML5

### Usage
1. Download a zip of this repository
2. Extract to a directory
3. Run `npm install` in that directory
4. Run `node app.js`
5. Connect to `http://your-ip:3000/`
